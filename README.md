# About

2023软件杯A3国二作品 算法19名

A 3D medical data analysis platform based on nnunet

# Framework

Vue3.0 + Flask


The 3D display is based on WebGL2.0.



# Start

You may need to start with
```Bash
npm run dev
```

You will also need a GPU server.
